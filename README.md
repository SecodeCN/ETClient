## ETClient版本说明
 1.这是基于ET 4.0 ET客户端的组件版本，适合于学习，或附加其他项目使用
 
 2.只供学习交流使用
 
 3.只有一个极小的开发包dll，导入项目即可使用，基于C# 4.5开发请确认项目开启了C# 4.X的支持
 
## ETClient运行使用方法
 1.使用Unity打开并导入组件，确认项目开启了C# 4.X的支持，运行Example下测试场景
 
 2.服务端版本保持参考源框架
